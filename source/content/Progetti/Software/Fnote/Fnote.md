- Software per prendere note super velocemente
- Con Win+R si apre "Esegui di windows". Tu scrivi lì la nota, metti i tag che ti interessano, e lui te la salva.
- Il comando tipico potrebbe essere qualcosa tipo
> fn /tag nota
- C'è già una versione molto molto "alpha" qua: [luca-deste/Fnote](https://github.com/luca-deste/Fnote)
- Con "fn /today" apri il software viewer sulla data di oggi.
- Devi poter filtrare nel software per data e per tag.
- I Tag devono essere configurabili dall'utente in un file .ini
- Devi poter usare /find per trovare una stringa nelle note e filtrare tutte le note con quella stringa.
- Il software deve essere utilizzabile senza installazione o deve poter essere installato solo come utente.
- Le note devono potersi "Auto taggare" usando le regex.
- Ai tag devono poter essere assegnati diversi colori, così che sia più facile farli saltare all'occhio.
- Nel caso un tag abbia più colori, può o mostrare entrambi, o mostrare il primo assegnato. Sarebbe il top se nella nota il colore fosse semplicemente un pallino, così da poterne far stare almeno 2 o 3.
- Deve esserci un viewer in cui si possano fare tutte le operazioni e si possano consultare le note.
- Dovrebbe essere possibile fare screenshot, e allegarli alla nota: _screenshot stile greenshot sarebbero il top_
- In una nuova versione, creare un menù contestuale dedicato. Io lo immagino a mo di chat, che appare in basso a destra sullo schermo una volta premuta la combinazione di tasti corretta, e dove posso scrivere e aggiungere la foto in uno storico di chat stile teams/whatsapp
